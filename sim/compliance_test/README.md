# Compliance Test

## What This Flow Does

This folder runs ACT4/Sail golden signature checks for the DUT.

Golden files are generated by a separate WSL-side repo:

[WsWSC/3-stage-riscv-golden-generator](https://github.com/WsWSC/3-stage-riscv-golden-generator)

## Files And Roles

Windows repo:

| Path | Role | Git |
| --- | --- | --- |
| `runner.py` | Compliance runner | committed |
| `test_one.py` | Run one compliance test | committed |
| `test_all.py` | Run all compliance tests | committed |
| `golden_sig/` | Sail golden signatures | generated |
| `DUT_metadata/` | Per-test metadata | generated |
| `DUT_runtime/` | DUT binaries/data/output | generated |

WSL generator repo:

| Path | Role |
| --- | --- |
| `~/risc-v/3stage-riscv-golden-generator/` | Golden generation scripts and ACT4 config |
| `~/risc-v/riscv-arch-test/` | Official ACT4 repo |
| `~/risc-v/tools/` | RISC-V toolchain and Sail |
| `~/risc-v/arch-test-compile/` | ACT4 workdir/cache |

## Generate Golden Files

Run from WSL:

```bash
cd ~/risc-v/3stage-riscv-golden-generator
cp config.env.example config.env
```

Edit `config.env`:

```bash
TARGET_REPO=/mnt/c/Users/<windows_user>/Documents/3-stage_RISC-V
```

Generate:

```bash
./generate_golden.sh
```

Generated output in this repo:

```text
golden_sig/*.sig
DUT_metadata/*.json
DUT_runtime/bin/*.bin
DUT_runtime/data/*.data
```

Expected count: `47` files in each output group.

## Run Compliance Tests

Run from Windows PowerShell at the repo root:

```powershell
python sim\compliance_test\test_one.py add
python sim\compliance_test\test_one.py beq
python sim\compliance_test\test_one.py sw
python sim\compliance_test\test_one.py div
```

Run all:

```powershell
python sim\compliance_test\test_all.py
```

Run tests one at a time. These scripts share `sim/inst_data.txt` and `sim/out.vvp`.

## Common Failures

| Message / Symptom | Fix |
| --- | --- |
| `missing compliance runtime binary` | Run the WSL golden generator. |
| `sail_riscv_sim` not found | Fix the generator repo `config.env` or WSL `PATH`. |
| `mise` not found | Install `mise` in WSL and reopen the shell. |
| Stale golden files | Regenerate golden files, then rerun the Windows test. |
